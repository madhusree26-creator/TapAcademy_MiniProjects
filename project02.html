📚 Tech Glossary

HTML
HTML (HyperText Markup Language) is the standard language used to create and structure web pages. It helps in designing webpage elements such as headings, paragraphs, images, links, and forms.
CSS
CSS (Cascading Style Sheets) is used to style and design web pages with colors, fonts, spacing, and layouts. It helps make websites attractive, responsive, and visually appealing to users.
JS
JavaScript is a programming language used to make web pages interactive and dynamic. It helps add features like animations, form validation, buttons, pop-ups, and real-time updates to websites.
